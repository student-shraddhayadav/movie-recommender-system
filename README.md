# 🎬 Movie Recommendation System

A Machine Learning based Movie Recommendation System that suggests similar movies based on user selection. Built using **Python, Streamlit, and Cosine Similarity**.

---

## 🚀 Features

* 🎥 Recommend movies based on similarity
* 🖼️ Displays movie posters using TMDB API
* ⚡ Fast and interactive UI using Streamlit
* 🧠 Uses Cosine Similarity for recommendations

---

## 🛠️ Tech Stack

* Python
* Pandas
* Scikit-learn
* Streamlit
* Requests API
